# Kwadjo Sarpong Aquaponics Website

This project showcases an interactive and educational website designed to highlight the concept of aquaponics and sustainable water systems. The website demonstrates a clean layout, multimedia integration, and responsive design principles using **HTML**, **CSS**, and **JavaScript**.

[View Project](https://kwadjosarpong.github.io/Kwadjo-Sarpong-Aqua-Website/)


## Objectives
- Create an engaging, visually appealing website that represents an aquaponics system  
- Practice front-end development skills through responsive web design  
- Integrate multimedia elements such as images and videos to enhance learning  
- Demonstrate clean code structure and organized styling conventions  

---

## Tools & Techniques
- **HTML5** for page structure and semantic layout  
- **CSS3** for design, layout control, and responsive behavior  
- **JavaScript (ES6)** for interactivity and dynamic page elements  
- **Media Integration** for embedding `aquaponicsVideo.mp4` and visual assets  
- **Version Control** using Git and GitHub  

---

## My Contributions
- Designed and developed the entire website layout from scratch  
- Implemented responsive design using CSS flexbox and media queries  
- Integrated and optimized video and image assets for performance  
- Wrote all HTML and JavaScript to ensure functionality and structure  
- Conducted cross-browser testing to ensure consistency across devices  

---

## Key Features
- Responsive design for desktop, tablet, and mobile viewing  
- Clean, modern UI with water-inspired visual theme  
- Embedded video showcasing an aquaponics setup  
- Modular and organized file structure for easy scalability  
- Lightweight and fast-loading static website  

---

## Limitations
The current version is static and does not include backend functionality or interactive form submissions. Future iterations could integrate:
- A contact form or information request system  
- Expanded content on sustainability and hydroponic systems  
- Additional animations or visual effects for user engagement
